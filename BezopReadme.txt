********************************Bezop Candidate Assignment************************************

This read me file is to enlist the instructions of the resources that make up this project.

1) The application uses Dropbox-v2-PHP-SDK to upload, download and remove files from dropbox

2) It uses AngularJS to pass user inputed data to PHP and vice versa.

3) All files and folders required to run this application can be found at https://github.com/OlammieConcept2/BezopGCP_Assignment

4) To add file, the AddFile.php upload the text file named TextFile.php in the same directory and also download the same file from Dropbox.
